# БЛОК I. Реальный главный итог: алгоритм комбинированного анализа

Этот блок должен быть финалом всех экспериментов.

## I1. Что есть сейчас

Сейчас в заключении уже есть рекомендация:

1. регрессия,
2. окна,
3. агрегаты памяти,
4. фазовые портреты,
5. осторожность с прогнозом. 

Это очень хорошо, но еще можно сделать сильнее.

---

## I2. Что нужно сделать

Собрать итоговую схему:

### Шаг 1. Диагностика окна/режима

По фазовому портрету, дисперсии, цикличности, вариативности.

### Шаг 2. Если режим раннего роста

Использовать прямую идентификацию параметров отображения + регрессию как контроль.

### Шаг 3. Если режим устойчивого информативного роста/цикла

Использовать ENTER/STEPWISE + Beta + (B_\Sigma).

### Шаг 4. Если плато

Не доверять точному списку лагов; опираться на ENTER, Beta, (B_\Sigma), прямые оценки потенциала.

### Шаг 5. Если крах/вырождение

Интерпретировать отсутствие модели как диагностический результат.

### Шаг 6. Для прогноза

Не использовать долгосрочный точечный прогноз в хаосе; ограничиваться горизонтом устойчивости или сценарной трактовкой.
