\chapter{Приложение Б. Расширенная линия промышленных индексов}
\label{app:prilozhenie-b-rasshirennaya-liniya-promyshlennyh-indeksov}

В основной текст глава 3 включает только краткую внешнюю проверку переносимости метода на промышленных индексах. Развёрнутые результаты по индексам промышленного производства вынесены в приложение Б.

В приложение Б входят:

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  Подробные сравнения трёх слоёв наблюдения (\texttt{adj\_smoothed}, \texttt{adj\_unsmoothed}, \texttt{raw}).
\item
  Полные таблицы режимов и интерпретируемости для всех окон по промышленному корпусу.
\item
  Отраслевые карточки, обзорные сводки по кейсам и дополнительные графики по шокам 2020 и 2022 годов.
\item
  Расширенные таблицы устойчивости маршрута анализа по промышленным рядам.
\end{enumerate}

Электронные материалы приложения Б соответствуют следующим наборам файлов:

\begin{itemize}
\tightlist
\item
  \texttt{outputs/real\_data/ipp\_routing\_all\_variants} --- режимы, интерпретируемость и выбор рабочего слоя наблюдения;
\item
  \texttt{outputs/real\_data/ipp\_case\_cards} --- расширенные отраслевые карточки, обзорные сводки по кейсам и материалы по шоковым эпизодам.
\end{itemize}
