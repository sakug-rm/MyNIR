\chapter{Приложение В. Дополнительные таблицы по линии выручки}
\label{app:prilozhenie-v-dopolnitelnye-tablitsy-po-linii-vyruchki}

Основной текст включает только те результаты по выручке, которые необходимы для доказательства центрального тезиса. Расширенные таблицы и дополнительные графики по группам и отраслям вынесены в приложение В.

В приложение В входят:

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  Полные таблицы состава групп фирм по темпам роста и соответствующие годовые медианные траектории.
\item
  Развёрнутые интервальные и оконные таблицы по группам выручки.
\item
  Полные секторные таблицы по отраслевому подкорпусу выручки, включая покрытия, устойчивость и шоковые профили.
\end{enumerate}

Электронные материалы приложения В соответствуют следующим наборам файлов:

\begin{itemize}
\tightlist
\item
  \texttt{outputs/real\_data/revenue\_b1} --- группы фирм по темпам роста, интервальные и оконные результаты;
\item
  \texttt{outputs/real\_data/revenue\_b2} --- отраслевой подкорпус выручки, карты читаемости и шоковые сводки.
\end{itemize}
