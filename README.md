# Дискретная динамика и AR-аппроксимации

Каркас реализации по ТЗ: симуляция логистического отображения, фазовый анализ, AR-модели (stepwise/enter), прогнозы и отчёт.

## Быстрый старт

1. Активируйте среду:
   ```bash
   source .venv/bin/activate
   ```
2. Запустите полный пайплайн (по умолчанию шаг сетки A/K = 0.25, прогноз 150 шагов):
   ```bash
   python experiment.py --step 0.25 --horizon 150 --n-steps 300
   ```
   Результаты сохраняются в `outputs/run_YYYYMMDD_HHMMSS/` (графики и `report.html`).

## Модули

- `simulation.py` — логистическое отображение, sweep по A/K, сплит ряда.
- `regression.py` — AR enter и stepwise (AIC/BIC), прогнозирование.
- `analysis.py` — фазовые точки, парабола, ошибки в фазовом пространстве.
- `visualization.py` — графики траекторий, фазовых портретов и прогнозов.
- `report.py` — простой HTML-отчёт.
- `experiment.py` — оркестрация полного сценария.

## Параметры `experiment.py`

- `--step` — шаг сетки по A/K (интервалы 0–1, 1–2, 2–3 + фикс. пары 2/2, 2.52/2.52, 2.8/2.8).
- `--horizon` — длина AR-прогноза (100–200 по ТЗ).
- `--segments` — число сегментов временного ряда (по ТЗ: 5).
- `--max-lag` — максимальный лаг AR (по ТЗ: 10).
- `--criterion` — критерий stepwise (aic или bic, по умолчанию bic).
- `--noise-std` — добавочный шум в симуляции.
- `--n-steps` — длина исходной траектории.
- `--output` — каталог для результатов.
- `--random-state` — сид для воспроизводимости.
